# AI Resume Screening System

## Live Demo
Streamlit App: https://rushirajyaguru-resume-screening-ai-app-qz7iaj.streamlit.app/

## Features
- Resume Classification
- Resume Matching
- Candidate Ranking
- PDF Resume Upload
- NLP-Based Text Processing
- Machine Learning Predictions

## Tech Stack

- Scikit-learn
- Streamlit
- Pandas
- PDFPlumber
